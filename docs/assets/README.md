# 文档图片资产

- `mi-ao-logo.png`：从米遥 VI 主标识精确提取的 1024×1024 透明底魔杖 Logo，用于 README 和后续品牌资产。
- `mi-ao-button-map.png`：米遥 MI-AO 在 macOS 上的默认按键示意图，用于中文与英文 README。

macOS App 使用同一魔杖标识的白色圆角方形版，1024 px 主文件位于 `Resources/AppIcon/AppIcon-1024.png`，构建时由 `scripts/build-icon.sh` 生成标准 `.icns`。

图中“菜单＝鼠标右键”表示小米蓝牙遥控器 2 Pro 连接 Mac 后的系统原生行为；米遥不重映射或拦截菜单键。其余功能说明以仓库当前代码和 [按键预设文档](../BUTTON_PRESETS.md) 为准。
