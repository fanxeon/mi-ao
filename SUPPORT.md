可复现软件问题请使用 Bug Report；新遥控器请使用 Device Support Request。请勿在公开 Issue 中上传私人语音、设备地址、peripheral UUID、序列、用户名或密钥。

For reproducible software problems, use the Bug Report template. For a new remote, use Device Support Request. Never upload personal speech, device addresses, peripheral UUIDs, serial numbers, usernames, or secrets to a public issue.

安全漏洞必须按 [SECURITY.md](SECURITY.md) / [Security Policy](SECURITY_EN.md) 私下报告，不要创建公开 Issue。
