# 米遥官网可使用的产品事实

基于米遥仓库 `main` 提交 `2e03aec` 与已安装 App `0.2.2`，官网可安全使用以下事实：

- 当前阶段：`source-first beta · V2 / 0.2.2`
- 平台：macOS 14+；当前不支持 Windows / Linux
- 已验证硬件：小米蓝牙遥控器 2 Pro，固件 2671
- 语音链路：遥控器自带 BLE 麦克风 → ATVV → ADPCM → 本地 Whisper → Codex
- 日常形态：安装后的原生米遥 App、设置与诊断窗口、菜单栏运行入口
- 真实设置：自动发送、遥控器按键控制、登录后启动、随时就绪 / 智能休眠
- 设备能力：扫描、选择、固定目标设备和确定性仲裁
- 按键能力：默认配置、自定义配置、热更新、真实高亮、单次测试、安全导入导出
- 安全边界：只有确认唯一可用 Codex 输入区才自动提交；否则使用本地回退路径
- 隐私边界：日常转写在本机完成，录音和文字记录保存在本机，不自动上传
- 分发边界：当前没有 Developer ID 公证 DMG；使用源码安装脚本构建并安装 App

不要使用以下声明：

- 支持所有蓝牙语音遥控器
- 零配置、绝不误发、百分之百识别
- 已提供正式签名 DMG 或自动更新
- 与小米、Google、OpenAI 或 Codex 联合开发或获得官方背书
- 没有真实测量依据的延迟、准确率和功耗数字
